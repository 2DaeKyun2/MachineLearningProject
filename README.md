# 대출 위험도 예측 모델
### 프로세스
* 데이터 탐색(EDA)
    - 가지고 있는 데이터로부터 통찰력을 얻는다.
* 라벨(답)의 존재 유무
    - 지도학습/비지도학습 결정
* 라벨의 형태
    - 분류/회귀 모델 결정
* Base-Line 모델로 가장 간단한 머신러닝 모형 구현
    - 아무런 결정없이 생성한 모델
    - EDA/데이터전처리 결과 확인을 위한 모델
    - 모델선택, 튜닝의 기준이 될 모델
    - Base-Line 모델의 문제점을 파악하여 그것을 개선하는 방향으로 튜닝해 나간다.
